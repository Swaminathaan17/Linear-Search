# Linear Search Algorithm

A simple implementation of the Linear Search Algorithm. Linear Search is one of the most basic searching techniques used to find an element in a collection by checking each item sequentially.

## Features

- Search for an element in an array/list
- Beginner-friendly implementation
- Works on sorted and unsorted data
- Easy to understand and modify
- Displays the position of the searched element

## How It Works

1. Start from the first element of the array.
2. Compare each element with the target value.
3. If a match is found, return its position.
4. If the end of the array is reached without finding the target, report that the element is not present.

## Example

Input:
Array: [10, 20, 30, 40, 50]
Target: 30

Output:
Element found at index 2

## Time Complexity

- Best Case: O(1)
- Average Case: O(n)
- Worst Case: O(n)

Linear Search is a fundamental algorithm that provides a strong introduction to searching techniques. While it may not be the most efficient for large datasets, it is simple, effective, and an important stepping stone toward learning more advanced algorithms such as Binary Search.
